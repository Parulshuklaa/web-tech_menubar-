# NLP Chatbot

A simple Python chatbot that uses basic NLP techniques:

- tokenization
- stop-word removal
- lightweight word normalization
- bag-of-words vectors
- cosine similarity for intent matching

## Run

```bash
python3 chatbot.py
```

## Example prompts

- `hello`
- `who are you`
- `can you help me`
- `what are your hours`
- `tell me about this portfolio`

## Customize

Edit the `INTENTS` list in `chatbot.py` to add new topics, patterns, and responses.
